Adding Tabs
-------------

Users have the option to add tabs to their dashboards to help organize content. To add tabs, first enter edit mode by clicking the edit button in the upper right-hand corner of the dashboard. The click the plus sign at the top to add a new tab. New tabs can be renamed by clicking on the tab name. By default the first tab is called "Main", but it can be renamed. Tabs can reference any of the information on other tabs.
